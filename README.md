# WhaleWatcher

## About / Synopsis
This application is for the express purpose of following the stock trades of wealthy individuals in congress that are requried to pubilcly post their trades. This is in ongoing development.
Currently we have the ability to search the House of Represenatives that trade on the stock market.

This will also be built out with various tools to find, pick, and purchase securities, crypto, and other investments
